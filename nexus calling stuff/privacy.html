üìä 1. Information We Collect

For Calling Features:

  • User ID: The unique identifier assigned to you by Discord, used to manage calls and cross-server communication. Stored only if reported
  • User Name, Server Nickname, and Display Name: Your Discord name is used during calls for identification purposes. This information is sent via webhooks and is only stored if a message is reported.
  • Avatar Image: Your Discord avatar is used during calls for identification purposes and to ensure no inappropriate avatar images are involved. It is also used to provide a good user experience. Stored only if reported

For Server Leaderboards:

  • Server Name and Server Image: For servers that appear in the top 10 most active servers leaderboard (accessible via the nleaderboard command), we collect and store the server name and server image/icon.
